Memory Game

Comment jouer?

Le but du jeu est de trouver toutes les paires d'images identiques.

Cliquez sur une carte pour la retourner et voir l'image cachée. Essayez de mémoriser l'emplacement de chaque image pour trouver les paires.

Le jeu est terminé lorsque toutes les paires d'images ont été trouvées.

Pour recommencer le jeu, cliquez sur le bouton "Commencer le jeu".
